\# 📰 Fake News Detector (ML Project)



This project uses Machine Learning to classify whether a news article is \*\*Fake ❌ or Real ✅\*\*.



\## 🚀 Features

\- Text classification using TF-IDF

\- Logistic Regression model

\- Command-line input

\- Real-time prediction



\## 🛠️ Tech Stack

\- Python

\- Pandas

\- Scikit-learn



\## ▶️ How to Run



1\. Clone the repository:

git clone https://github.com/your-username/fake-news-detector-ml.git



2\. Navigate to folder:

cd fake-news-detector-ml



3\. Install dependencies:

pip install -r requirements.txt



4\. Run the project:

python app.py



\## 🧪 Example Output



Enter news text: Win money instantly by clicking this link  

Output: ❌ FAKE  



Enter news text: Government launches new policy  

Output: ✅ REAL
